## Set your theme

```
import { applyTheme } from '@velumweb/ui-kit';

applyTheme({
    '--primary-btn-color': '#7C3AED',
    '--secondary-btn-color': '#FFFFFF',
    '--hover-btn-color': '#6D28D9',
    '--bg-color': '#fff',

    '--corners-standard': '0.65em',
    '--corners-round': '3em',
    '--corners-square': '0',
});
```
